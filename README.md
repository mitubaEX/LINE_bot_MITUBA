# LINE_BOT_MITUBA

## Usage
Please write your token to ./env_var.txt.

And run the following command.

```
sh setup.sh
```

Please open other window, and ngrok command.

```
ngrok http 8000
```

## dependencies

- [line/line-bot-sdk-python](https://github.com/line/line-bot-sdk-python)
- docker
- python3.6.4
